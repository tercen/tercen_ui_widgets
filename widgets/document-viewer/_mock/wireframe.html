Document Editor — Layout Wireframe

Skeleton / annotation only. No design tokens applied. All dimensions in px from window_constants.dart & app_spacing.dart.

Layer Dimensions

Tab strip→h=32px (tabStripHeight). Tabs: h=28px, r=6px top, maxW=220px
Toolbar→h=48px (headerHeight). Buttons: 32×32px, r=8px, icon=16px. Gap=8px (sm)
Toolbar buttons→Save (leading), Mode toggle (segmented 2), formatting groups, spacer, Print (trailing)
Divider→h=1px (lineSubtle). Separates toolbar from body
Body→Expanded (flex:1). Routes to loading / active / error state
Min widget width→208px (5×32 + 4×8 + 2×8)
Body state centred→maxWidth=280px for error/loading. Spinner=28×28px, stroke=3px
Narrow threshold→<680px: formatting buttons collapse to single [Format] icon-only button
Source editor→monospace 14px, lh=1.6, pad=16px, no border, fill=surface
Rendered editor→flutter_markdown, pad=16px, fill=surface, selectable, links open _blank
Banner→Full width, py=6px px=16px, icon=14px + gap=8px + text=12px
